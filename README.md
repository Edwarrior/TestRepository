# TestRepository
A test repository for practising key GitHub features &amp; python coding.
It would be good to try GitHub fetching, commiting, pushing, and merging. Additionally, raising issues and pull requests.


# Getting started
* Send over your account name and I'll add you as a collaborator
* I recommend that you install GitHub Desktop
* Clone the project by clicking open with GitHub Desktop on the github page

<img width="561" height="435" alt="image" src="https://github.com/user-attachments/assets/3d8cea94-ec8d-46a1-b36f-5bb130c94e7f" />

Any changes you make to your local folder where the repository contents is kept will be recorded by GitHub Desktop. Any changes can be commited through the Desktop app.
